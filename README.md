# Pycord Template by Starcea

## Setup venv and necessary files

```bash
$ cp .env.example .env
$ touch "your database file"
$ python3 -m venv venv
$ . venv/bin/activate
$ pip install -Ur requirements.txt
```

## Setup .env

```py
TOKEN="your token"
DATABASE="your database file"
```

## Enjoy!

```bash
$ python main.py
```
